Screenshots folder (placeholder)
================================

This folder is a placeholder for screenshots if you later add them. If you don't have screenshots, leave this folder in the repo as-is — judges will still accept commit/diff evidence.

Suggested filenames:

- `01-dashboard.png`
- `02-beneficiaries.png`
- `03-opportunities.png`
- `04-matching.png`
- `05-persistence-proof.png`

How to add screenshots later:

1. Take screenshots on macOS (Cmd+Shift+4) and save them with the names above into this folder.
2. Add image links to your README or submission post:

```
![Dashboard](docs/screenshots/01-dashboard.png)
```

If you prefer not to add images, keep this folder and point judges to `docs/evidence.md` for verification steps.
